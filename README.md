# 자바 퍼즐게임（パズルゲーム，PulzzeGame）



Java로 제작한 간단한 퍼즐게임입니다. （JAVAで作った簡単なパズルゲームです。）

난이도는 3x3 , 4x4 의 난이도를 제공하고 있습니다.（難しさは３X3、４X４二つを提供しています。）
